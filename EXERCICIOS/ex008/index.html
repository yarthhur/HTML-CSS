Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag b (forma não semantica)

Nesta frase, tenho um termo em destaque usando a tag strong (forma semantica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag i (não semantico)

Nesta frase temos um termo em ênfase usando a tag em (forma semantica)

Texto marcado

Podemos criar também um texto marcada usando a tag mark

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo usando a tag big e small (big não é semantica, não use) (se quiser aumentar a letra use o CSS)

Texto deletado

Podemos marcar um texto como excluído ou deletado, para indicar que ele deve ser lido, mas não considerado, usando a tag del

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adcionado depois, usando a tag ins

Texto sobrescrito

Para inserir coisa do tipo x20+3, usando a tag sup

Texto subscrito

Para inserir coisas do tipo H20, usando a tag sub
